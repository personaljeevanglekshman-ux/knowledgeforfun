# 🌿 KnowledgeForFun

A modern, responsive personal blog website built using **HTML**, **CSS**, and **JavaScript**. The website is designed to showcase photography, nature, architecture, and storytelling through an elegant and interactive user interface.

---

## 📖 About

**KnowledgeForFun** is a visually engaging blog where every photograph tells a story. It combines beautiful imagery with informative content, allowing visitors to explore nature, architecture, and photography while learning the history and significance behind each subject.

This project was built as a front-end web development project using only vanilla HTML, CSS, and JavaScript without any frameworks.

---

## ✨ Features

- 🌿 Modern and responsive design
- 🌙 Dark/Light mode with theme persistence
- 📱 Mobile-friendly navigation menu
- 🎯 Smooth scrolling navigation
- 🖼️ Interactive image gallery with lightbox
- 📚 Featured article cards
- ✨ Scroll-triggered animations using Intersection Observer
- ⌨️ Hero section typing animation
- ⬆️ Scroll-to-top button
- 📩 Newsletter subscription form
- 📬 Contact form
- 🔄 Dynamic footer year
- 🎨 Clean UI with glassmorphism effects and smooth transitions

---

## 🚀 Getting Started

1. Clone the repository

```bash

```

2. Navigate to the project folder

```bash

```

No installation or additional dependencies are required.
